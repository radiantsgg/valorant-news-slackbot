ReconBolt - Valorant News Bot

A Slack bot named ReconBolt that sends you spicy Valorant news from playvalorant.com

The team at radiants.gg wanted to get notified of all the latest Valorant news, that is why we created ReconBolt. Built as an open source project you can install it with the button below or deploy your own version with the instructions on Github. Please send the team your feature or issue requests!
